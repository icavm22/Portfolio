# Portfolio de Antonio Victoriano

Portafolio profesional de Antonio Victoriano, especialista en Business Intelligence y consultoría empresarial.

## 🚀 Características

- ✨ Diseño moderno y profesional
- 🎨 Animaciones suaves con Framer Motion
- 📱 Totalmente responsive
- ⚡ Construido con Next.js 14 y TypeScript
- 🎯 Optimizado para SEO
- 💫 Experiencia de usuario excepcional

## 🛠️ Tecnologías

- **Framework**: Next.js 14
- **Lenguaje**: TypeScript
- **Estilos**: Tailwind CSS
- **Animaciones**: Framer Motion
- **Iconos**: React Icons

## 📦 Instalación

```bash
npm install
```

## 🏃 Desarrollo

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## 🏗️ Construcción

```bash
npm run build
npm start
```
